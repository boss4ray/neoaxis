# NeoAxis

Entertainment Production Company.

Design principles:

- NeoAxis is an entertainment production company, not a single-universe website.
- Each Work is an independent intellectual property.
- Every Work can have its own Story, Music, Gallery, and Assets sections.
- Components, styles, scripts, and shared images live under `shared/`.
- Top-level navigation stays limited to HOME, WORKS, MUSIC, and ABOUT.
- New works can be added by creating a folder under `works/`.

Each content page has its own URL, title, description, and canonical link for SEO.
